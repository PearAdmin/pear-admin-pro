### 开发任务

1. 顶部布局
2. 侧边布局
3. 联动布局
4. 多选项卡
5. 权限处理
6. 登录逻辑
7. 主题配置
8. 夜间模式

### 项目安装
```
npm install
```

### 项目运行
```
npm run serve
```

### 编译项目
```
npm run build
```
